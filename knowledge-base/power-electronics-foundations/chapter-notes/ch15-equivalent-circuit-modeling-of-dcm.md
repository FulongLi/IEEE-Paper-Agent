# Chapter 15: Equivalent Circuit Modeling of the Discontinuous Conduction Mode

## Retrieval Tags

DCM modeling, equivalent circuit, sampled dynamics, load-dependent gain, small-signal DCM

## Core Ideas

- DCM requires different equivalent models because energy transfer includes a zero-current interval and load-dependent conversion behavior.
- Small-signal DCM models often differ in order and dynamic structure from CCM models.
- Correct DCM modeling is essential for light-load regulation, mode transition analysis, and control design.

## Useful IEEE Language

- "The DCM equivalent circuit accounts for the zero-current interval and the resulting load-dependent conversion characteristic."
- "Because the operating mode changes the dynamic order of the converter, the controller must be evaluated across the expected load range."
