# Chapter 18: Current Programmed Control

## Retrieval Tags

current-mode control, peak current control, slope compensation, inner current loop, subharmonic oscillation

## Core Ideas

- Current programmed control uses an inner current loop to shape power-stage dynamics and improve disturbance response.
- Peak current-mode control can simplify outer-loop design but introduces sampling effects and potential subharmonic oscillation at high duty ratios.
- Slope compensation and accurate current sensing are critical for stable implementation.

## Useful IEEE Language

- "Current programmed control introduces an inner current loop that modifies the effective power-stage dynamics."
- "Slope compensation is included to suppress subharmonic oscillation under high-duty-ratio operation."
