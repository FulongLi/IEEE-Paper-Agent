# Chapter 21: Pulse-Width Modulated Rectifiers

## Retrieval Tags

PWM rectifier, active front end, power factor correction, grid current control, DC-link regulation

## Core Ideas

- Pulse-width modulated rectifiers actively shape input current while regulating a DC output or DC-link voltage.
- Compared with passive rectifiers, PWM rectifiers can improve power factor, reduce harmonic distortion, and support bidirectional power flow in suitable designs.
- Control design must coordinate grid synchronization, current regulation, DC-link dynamics, and switching constraints.

## Useful IEEE Language

- "The PWM rectifier regulates the DC-link voltage while shaping the input current to reduce low-order harmonics."
- "The active front-end control strategy improves power factor and enables controlled power flow between the AC source and DC bus."
