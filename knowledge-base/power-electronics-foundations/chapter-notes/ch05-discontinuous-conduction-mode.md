# Chapter 5: The Discontinuous Conduction Mode

## Retrieval Tags

DCM, CCM boundary, inductor current, conversion ratio, light-load operation, nonlinear dynamics

## Core Ideas

- Discontinuous conduction mode occurs when inductor current reaches zero within a switching period.
- In DCM, the conversion ratio depends on load, inductance, switching frequency, and duty ratio, so the converter behaves differently from CCM.
- DCM can reduce certain losses at light load but complicates modeling, control, and regulation.

## Useful IEEE Language

- "At light load, the converter may enter discontinuous conduction mode, where the inductor current contains an interval of zero current."
- "The DCM conversion characteristic is load dependent, which must be considered in controller design and operating-range analysis."
