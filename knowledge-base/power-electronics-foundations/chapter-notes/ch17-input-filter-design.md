# Chapter 17: Input Filter Design

## Retrieval Tags

input filter, EMI filter, source impedance, negative input resistance, stability interaction

## Core Ideas

- Input filters reduce conducted noise and source ripple but can interact with converter input impedance and destabilize the system.
- Filter design must satisfy attenuation requirements while preserving acceptable converter stability margins.
- Damping and impedance shaping are often required to manage filter-converter interaction.

## Useful IEEE Language

- "The input filter is designed to attenuate conducted disturbances without degrading the converter stability margin."
- "Filter-converter interaction is evaluated by comparing the filter output impedance with the converter input impedance."
