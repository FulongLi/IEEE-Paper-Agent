# Chapter 16: Techniques of Design-Oriented Analysis - Extra Element Theorems

## Retrieval Tags

extra element theorem, design-oriented analysis, impedance interaction, sensitivity, added component

## Core Ideas

- Extra element theorem methods evaluate how adding or modifying an element changes a transfer function or impedance.
- The approach is useful when a converter model is already known and a parasitic, filter, sensor, or compensation element must be included.
- Design-oriented factorization helps identify which added elements are performance critical.

## Useful IEEE Language

- "The extra element theorem is used to isolate the influence of the added network on the nominal converter response."
- "This representation clarifies how the parasitic element modifies the pole-zero structure."
