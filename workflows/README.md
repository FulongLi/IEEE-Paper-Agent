# Workflows

This folder contains reusable workflows for turning a research idea into an IEEE-style paper artifact.

## Main Workflows

- `idea-to-paper.md`: Convert a raw research idea into a paper plan, outline, and evidence roadmap.
- `paper-production.md`: Move from a validated paper plan to LaTeX draft production.
- `prompt-library.md`: Copy-ready Codex prompts for common paper-writing tasks.

## Recommended Starting Prompt

```text
Use this repository as an IEEE paper-writing assistant. Follow AGENTS.md and workflows/idea-to-paper.md. My research idea is: <paste idea>. Help me turn it into a publishable paper plan.
```
